TownScan

Während meiner Studienzeit in Zürich suchte ich mehrere Male eine WG ausserhalb der Stadt, wollte aber gleichzeitig mit den öffentlichen Verkehrsmitteln eine möglichst kurze Anfahrtszeit zur Uni. Deshalb musste ich für jede ausgeschriebene WG die Transsportzeit auf www.sbb.ch nachschlagen, um abzuschätzen ob die jeweilige WG für mich überhaupt in Frage käme.

TownScan automatisiert diese Aufgabe: Es liest eine Datei mit verschiedenen Ortschaften, sowie einen Zielort und einen Abfahrtszeitpunkt ein. Die Reisedaten von jeder Ortschaft zum Zielort werden von der Website fahrplan.search.ch mithilfe von urllib2 heruntergeladen und danach mittels regulären Ausdrücken (re) die Reisezeiten extrahiert.

Mithilfe dieser Informationen sortiert TownScan die eingegebene Liste der Ortschaften aufsteigend nach Reisezeit und speichert sie in einer Datei, so dass man auf einen Blick sieht ob eine gegebene Gemeinde eine vergleichsweise gute Verkehrsanbindung zu einem vorbestimmten Zielort hat.
