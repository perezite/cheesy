Dedicated Server Web User Interface

Im Rahmen meiner Tätigkeit als Junior Software Engineer bei GIANTS Software GmbH in Schlieren wär ich für die Entwicklung eines Dedicated Server Web User Interface für den Farming Simulator 2013 verantwortlich. Mein Aufgabenbereich umfasste die Erweiterung eines servlet-basierten HTTP Webservers und darauf aufbauend die Implementierung des Web UIs. Die komplette Software ist in C++ geschrieben.

Die Dedicated Server Software erlaubt es, eine Multiplayer-Instanz des Spiels Farming Simulator 2013 auf einer Server-Maschine (zum Beispiel in einem Datenzentrum) laufen zu lassen und über das Web UI zu steuern.


Features des Web UI:

  • Integrierte User-Verwaltung für feinkörnige Kontrolle der Zugriffsrechte
  • Konfiguration, Start, Stopp und Monitoring der laufenden Multiplayer-Instanz.
  • Up- und Downoad sowie automatische Installation von Savegames und Spiel-Modifikationen.
  • Logbuch zur Protokollierung des Spielfortschritts
  • Automatischer Neustart der Multiplayer-Instanz zu einem gesetzten Zeitpunkt
  • XML-Webfeed des Spielfortschritts zur Integration in ein Forum oder eine Website
  • Anzeige der Webserver- und Game-Logs
