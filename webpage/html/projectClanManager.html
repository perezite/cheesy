ClanManager

Nethour (2005 übernommen von Computerline GmbH) war ein Schweizer Hosting-Provider, welcher unter anderem im Hosting von Gameservern tätig war. Aufgrund des wachsenden Kundenstammes beauftragte mich Nethour 2004 mit der Umsetzung des Kundenadministrations-Tools ClanManager.

Die Kunden werden unterteilt in Spieler-Teams (sogenannte Clans), welche einen oder mehrere Server mieten können. ClanManager erleichterte die Administration der Clans und ihrer Server in einem einfachen PHP WebGUI. Ebenso bietet das Tool eine simple Rechnungsverwaltung.

Kunden- und Rechnungsdaten werden in einer MySQL Datenbank gespeichert.
